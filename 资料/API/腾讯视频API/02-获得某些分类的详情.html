开发文档

http://mobile.video.qq.com/fcgi-bin/getjimudata

跳转到： 导航, 搜索

获得某些分类的详情

HTTP请求方式

GET

请求参数

  类型及范围 说明
type int 0代表获得子类别id，1代表获得子类别的详细数据
platform int 7
version int 版本号。首页用31000，热门用33100
ztid int 类型id
leafids string 子类型id，多个子类型之间用+拼接。当type=1的时候才有用
otype string json或者xml，默认是xml
